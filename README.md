# Manoj L - Professional Portfolio

This is a premium, high-performance developer portfolio built with **React.js**, **Vite**, and **Framer Motion**.

## 🚀 Key Features
- **Modern UI/UX**: Glassmorphism design system with neon accents and fluid gradients.
- **Dynamic Animations**: Scroll-triggered reveals and interactive hover effects.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
- **Performance Optimized**: Built for speed and SEO.

## 🛠️ Technology Stack
- **Framework**: React.js 18 (Vite)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Premium Vanilla CSS

## 📦 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 📸 Personalization
To add your own image:
1. Replace the `PROFILE_IMAGE` URL in `src/App.jsx` with your local image path or a direct link.
2. Update the `projects` array in `App.jsx` to links your live projects.
